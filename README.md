# E-shopping Platform 

## Overview

E-shopping Platform is a full-stack e-commerce web application developed using the MERN Stack (MongoDB, Express.js, React.js, and Node.js). The application provides a modern online shopping platform where users can browse products, search for items, manage their shopping cart, place orders, and submit product reviews. The system is designed to deliver a seamless and responsive shopping experience while maintaining secure and efficient data management.

---

## Key Features

### User Features

- User Registration and Login Authentication
- Browse and Search Products
- View Product Details
- Add Products to Shopping Cart
- Update Cart Quantity
- Submit Product Reviews and Ratings
- View All Reviews of a Product
- Place and Manage Orders
- Responsive User Interface
- Secure User Account Management

### Admin Features

- Create New Products
- Update Existing Products
- Delete Products
- Manage Customer Orders
- Manage Registered Users
- Monitor Product Reviews
- Dashboard Management

---

## Product Review Functionality

The application includes a product review and rating system that allows authenticated users to submit feedback on purchased products. Reviews are stored in MongoDB and displayed on the product details page. Customers can view all submitted reviews, helping them make informed purchasing decisions based on previous user experiences.

## Installation Guide

### Clone Repository

bash
git clone https://github.com/Cod3shield/E-shopping-Platform


### Install Frontend Dependencies

bash
cd frontend
npm install


### Install Backend Dependencies

bash
cd backend
npm install


### Run Backend Server

bash
npm run dev


### Run Frontend Application

bash
npm start
